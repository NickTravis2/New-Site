About

My name is Nick Travis and I am a Full-Stack Web Developer based out of San Diego, California! Collaborating with talented people to create highly-impacting and user-friendly applications is something I'm incredibly passionate about. My curiosity drives me to continuously learn and try new things both inside and out of the development environment.

The website is designed to showcase my understanding of developing modern UIs as well as applications for the web. Scroll effects were created with the help of Parallas.js. General website design was done using HTML5, CSS3, and Javascript. The landing page background was created using Adobe Photoshop and the forms/comment section use PHP on the back-end via the the XAMPP Stack.

Projects added to the site include: 'Photo portfolio' using Flexbox & lightroom.js, 'Log-in system' and 'Comment section' using PHP, MySQL, AJAX & jQuery. The 'Arcade' section was created in HTML Canvas running on PHP & Javascript. The site is also fully responsive on most devices. Please mention any issues you discover in the feedback section below. Thanks for checking it out!
